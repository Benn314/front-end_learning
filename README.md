#### 小技巧合集

1. 非热键下 F2 重命名文件/文件夹 在vscode依旧适用 配合鼠标自定义编程键 标记md笔记代码块文件名会很快
2. **解决vscode gitpush文件没有回车换行效果**
   1. **原因**：github上的 `md`文件 默认是没有换行效果的（github服务器把换行效果解析了）
   2. **解决**：在本地md文件中  直接在要换行的语句最后打上 `Tab` 键（也就是4个空格） 就可以实现换行效果
